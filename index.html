Editor de Imágenes

Instrucciones de uso
  1. Arrastra una imagen al recuadro o usa el botón para subir una imagen base.
  2. Selecciona una imagen de gafas del catálogo para superponerla.
  3. Ajusta la posición, tamaño y rotación de las gafas según desees.
  4. Haz clic en "Descargar Imagen" para guardar tu creación.
